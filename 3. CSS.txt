1. 가상 선택자 (pseudo selectors)
    1) 선택한 요소가 특별한 상태여야 만족할 수 있음
    2) 구분
        - 동적 가상 선택자
        - 구조적 가상 선택자

2. 동적 가상 선택자 (dynamic ~)
    1) 어떤 상태나 조건이 발생할 떼,
       사용자의 액션에 따라 스타일이 바뀌는 선택자이다.
    2) active, visited, disabled, hover, focus
        - active
            - 클릭 시 활성화되는 가상 클래스
                <style>
                적용 대상 태그
                    --
                    a:active    { color : red }
                      ------
                      가상선택자
                </style>
        - visited
            -사용자가 이미 방문한 링크를 표시해준다.
        - disabled  
            - 비활성화된 요소를 나타냄
            - 비활성화를 시키는 역할을 한다
            - 단독으로 쓰이기 보단 조건을 추가해 조건에 해당하지 않는 경우
              폼에 입력할 수 없도록 만드는 용도로 사용됨
        - hover
            - 마우스 커서를 요소에 올려놓았을때 선언한 스타일을 실행함
        - focus 
            - input으로 만든 폼 등 집중(focus)을 받은 요소를 나타냄
            - pc에서는 마우스 클릭이나 tab키를 클릭할 때 발동함

    3) 구조적 가상 선택자(structural ~)
        1) 종류 : first-child, last-child, nth-of-type ...
        2) first-child  
            - 형제 중 첫번째(맏이)요소 선택하기
            - 반드시 부모 요소를 가지고 있어야 함
        3) last-child   
            - 형제 중 마지막(막내) 요소 선택하기
        4) nth-of-type
            - 형제 위치 기반 요소 선택하기
            - 형제 그룹 안에서의 위치
            - n, n+1

4. 애니메이션 
    1) 전환(transition)
        - 시간에 따른 상태를 지정하는 애니메이션임.
        - 형태 변형을 부드럽게 해줌.
        - 전환 속성, 전환 시간, 전환 타이밍(linear, ease, ease-in, ease-out, ease-in-out), 전환 지연
        -  EX.
            - 색이 변하는 사각형
                - 마우스를 사각형 위에 올려두면 변함

    2) 변형(transform)
        - 외적인 형태의 변형을 줄 수 있음
        - 전환과 함께 사용 -> 부드러운 애니메이션 만들 수 있음
        - 회전
            - 지정된 각도(deg)만큼 요소를 시계 방향 또는 반시계 방향으로 회전시킴 
        - 기울이기
            - x축과 y축에 제공된 값에 따라 기울인다.
        - 크기
            - 요소 크기를 늘리거나 줄임
            - 
        - 이동 효과    
        - x축, y축, z축
